= Spring Data Meilisearch
Junghoon Ban
:revnumber: {version}
:revdate: {localdate}
:feature-scroll: true

_Spring Data Meilisearch provides Meilisearch connectivity and repository support for the Meilisearch search engine.
It eases development of applications with a consistent programming model that need to access Meilisearch indexes._
