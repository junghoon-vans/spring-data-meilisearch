[[meilisearch.reference]]
= Meilisearch
:page-section-summary-toc: 1

This chapter describes the store-specific integration provided by Spring Data Meilisearch: client bootstrap, object mapping, operations, index lifecycle management, and index settings.

The main integration layers are:

* xref:meilisearch/client-configuration.adoc[Client configuration] for JavaConfig and XML namespace bootstrap.
* xref:meilisearch/object-mapping.adoc[Object mapping] for `@Document`, mapping metadata, and conversion.
* xref:meilisearch/template.adoc[Operations] for document and search APIs through `MeilisearchOperations`.
* xref:meilisearch/index-operations.adoc[Index operations] for instance and index lifecycle APIs.
* xref:meilisearch/settings.adoc[Settings] for annotation-driven and runtime index settings.
